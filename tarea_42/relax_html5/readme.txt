Proyecto HTML en el que se ha creado un proyecto básico integrando HTML con CSS y Javascript.
También se ha incluído un vídeo que se puede controlar por voz: hay que decir "yes" para reproducir el vídeo y
"no" para pausarlo.
La librería utilizada para control de voz es  Annyang
("//cdnjs.cloudflare.com/ajax/libs/annyang/2.6.1/annyang.min.js")
El navegador tiene que tener soporte para el control por micrófono para que funcione.
Se ha utilizado para probar Chrome (Version 93.0.4577.63)
